# Inventory System

A simple Python inventory management application that stores products in a local dictionary.

## Products

| Category | ID      | Name        | Price  | Quantity |
| -------- | ------- | ----------- | ------ | -------- |
| Pants    | pants01 | Blue Pants  | 32.000 | 30       |
| Shoes    | shoes01 | Brown Shoes | 23.000 | 50       |
| Hats     | hat01   | Black Hat   | 15.000 | 300      |

## Features

1. Show product categories
2. Show product details
3. Add new products
4. Search products by ID
5. Exit the application

## Usage

Run the program and select an option from the menu. The inventory data is stored in memory and resets when the program is closed.
